 CNN Keras Image Classification Demo

# Overview
This project demonstrates a simple Convolutional Neural Network (CNN) built using Keras.  
It is designed for learning purposes to understand how CNN architectures are structured.

The model is configured for binary classification and includes key deep learning components such as convolution layers, pooling, dropout, and fully connected layers.

---

#  Model Architecture

- Conv2D (64 filters, 3x3 kernel, ReLU activation)
- MaxPooling2D (2x2)
- Conv2D (128 filters, 3x3 kernel, ReLU activation)
- MaxPooling2D (2x2)
- Flatten layer
- Dense layer (128 units, ReLU)
- Dropout (0.5)
- Output layer (Sigmoid activation)

---

#  Technologies Used

- Python
- Keras (TensorFlow backend)
- NumPy
